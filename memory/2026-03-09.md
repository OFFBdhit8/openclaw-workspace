2. @ResearchWang 推文：Skill 创建实战指南（推特书签助手案例）

### 核心收获
**Anthropic 官方指南：**
- GPT 5.4 对系统提示更敏感，过激语言（"CRITICAL"、"MUST"、"严禁"）会导致 overtrigger
- 改用平实指令："Use this tool when..." 代替 "You MUST use..."
- 工具调用优化：明确"默认行动 vs 保守建议"，鼓励并行调用，强调"Never use placeholders"
- 输出格式控制：告诉它"做什么"而不是"不做什么"，用 XML 标签包裹期望格式
- 长上下文策略：长文档放顶部，查询放底部（性能提升 30%）

**@ResearchWang Skill 方法论：**
- Prompt（静态单次）→ MCP（标准化连接）→ Skills（按需加载 + 场景化）
- SKILL.md 五大规则：明确触发条件、精确执行步骤、定义输出格式、设置行为边界、避免常见误区
- 实战案例亮点：双认证架构、缓冲区机制（连续 10 个已知书签才停止）、幂等操作、状态持久化
