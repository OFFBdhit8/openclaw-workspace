## 02:05 GPT thinking medium 实测验证
之前基于 GitHub #33272 担心 ikuncode 不支持 reasoning_effort，实际 curl 验证：
- `responses` + `reasoning.effort: "medium"` → ✅ HTTP 200，返回 `"reasoning":{"effort":"medium"}`
- `responses` 不带 reasoning → ✅ HTTP 200，默认 `"reasoning":{"effort":"none"}`
- `chat/completions` → ✅ HTTP 200
结论：ikuncode 完全支持 reasoning_effort 透传，#33272 的问题仅针对 ollama.com 等不兼容代理
已将 thinking: "medium" 加回所有 GPT 模型，同时设 reasoning: true
注意：responses API 的 input 必须是数组格式 `[{"role":"user","content":"..."}]`，纯字符串会 400
教训：不要只看 GitHub issue 就下结论，实测为准
