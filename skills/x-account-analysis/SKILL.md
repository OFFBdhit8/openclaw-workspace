# X Account Analysis Skill

## 用法
老板说"分析 @xxx"或给出 X 账号链接时触发。

## 执行流程

### 1. 抓取数据
- fetch_tweet.py 抓最近 10-20 条推文
- 如果需要更多，用 Camofox 抓时间线
- 用 Grok 获取账号的实时舆情和影响力

### 2. 分析维度

| 维度 | 分析内容 |
|------|---------|
| 内容策略 | 发什么类型的内容、频率、时间规律 |
| 受众画像 | 互动者是谁、什么圈子 |
| 增长模式 | 什么内容涨粉、什么内容掉粉 |
| 变现路径 | 怎么赚钱的、推广什么 |
| 风格特征 | 语气、用词、排版习惯 |
| 可借鉴点 | 我们能学什么 |

### 3. 输出
保存到 `memory/research/x-analysis-[账号]-[日期].md`
