# Content Factory Skill

## 流程

### 1. 选题确认
- 老板给话题，或从 weekly-research 结果中选
- 确认：平台、受众、目的（引流/变现/品牌）

### 2. 调研（2分钟）
- web_search 抓相关热点
- 看竞品怎么写的（X/Reddit/知乎）
- 找数据支撑

### 3. 写作
根据平台适配风格：

| 平台 | 风格 | 长度 |
|------|------|------|
| X/Twitter | 短平快，hook 开头，数据说话 | 280字内 |
| 知乎 | 专业深度，有理有据 | 1000-3000字 |
| 公众号 | 故事化，有温度 | 1500-2500字 |
| Reddit | 真实经验分享，不营销 | 500-1500字 |
| Discord | 社区风格，互动感 | 200-500字 |

### 4. 优化
- 标题 A/B 测试（给 3 个标题选项）
- SEO 关键词（如果是长文）
- CTA（行动号召）

### 5. 输出
- 保存到 `memory/content/[平台]-[日期]-[标题].md`
- 老板确认后再发布（对外操作先确认）

## 批量模式
老板说"围绕 [话题] 出 5 条推"时，批量生成，风格递进。
